# MoonLens

A lightweight screen translation tool for Windows. Capture any area of your screen and instantly translate the text.

## Features

- **Screen Capture & OCR** — Select any area and extract text automatically
- **Real-time Translation** — Powered by Google Translate
- **Multiple Languages** — English, Spanish, Japanese, Chinese, Korean
- **Keyboard Shortcuts** — Customizable hotkeys for quick access
- **Dark/Light Mode** — Choose your preferred theme
- **Background Mode** — Floating popup when app is minimized
- **Bilingual Interface** — English and Spanish UI

## Requirements

- Windows 10/11
- Internet connection
- [OCR Language Packs](ms-settings:regionlanguage) for source languages

## Installation

1. Download the latest release
2. Extract to your preferred location
3. Run `moonlens.exe`

## Usage

1. Select source and target languages
2. Click **Select Area** or use your keyboard shortcut
3. Draw a rectangle around the text you want to translate
4. View the translation in the app or floating popup
